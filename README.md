# Better in Jeans

Better in Jeans is a small, honest space built on the belief that effort and reciprocity still matter.  

The name comes from a simple truth: most things feel more real and approachable when you are unpretentious, and willing to show up as yourself — better in jeans.

## What This Site Is

A Jekyll-powered blog hosted on GitHub Pages, designed to be:

- **Approachable** — no polish for its own sake  
- **Unpretentious** — denim over pretense  
- **Hardworking** — built piece by piece, post by post  
- **American‑classic** — simple, durable, intentional  
- **Reciprocal** — grounded in the idea that what you give comes back  

## Philosophy
This project is built on a simple idea: What we give when we give without expectation and yet how that returns to us.

## Structure

The site uses a clean, minimal directory layout:
.
├── _config.yml
├── _layouts/
├── _includes/
├── _posts/
├── assets/
├── index.md
├── about.md
├── now.md
├── archive.md
├── tags.md
└── 404.md


Posts live in `_posts/`.  
Pages like **About**, **Now**, and **Archive** sit at the root for clarity.
